Загрузка лог-файла для анализа

Поддерживаемые форматы: .log, .txt, .json, .csv (макс. 16MB)
Получить бесплатный API ключ | Оставьте пустым для работы в тестовом режиме
Что происходит после загрузки:
  1. Извлекаются все IP-адреса и SHA256 хэши
  2. Каждый артефакт проверяется через VirusTotal API
  3. Формируется отчет с результатами проверки
  4. Вы получаете детальный отчет для скачивания
